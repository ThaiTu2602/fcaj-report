### Week 2 Achievements:

* **Completed Onboarding and Received Credits**: Successfully completed the onboarding tasks and received 100$ AWS Credits to support learning and hands-on practice costs on the Cloud platform.
* **Established Cloud Cost Management**: Configured AWS Budgets with custom alert thresholds, enabling cost control and timely notifications when spending exceeds the projected budget.
* **Mastered AWS Technical Support Process**: Clearly understood AWS Support plans and learned how to create Support Cases to receive assistance when encountering technical issues with AWS services.
* **Implemented Group-Based Access Management**: Created IAM Groups and assigned IAM Users to groups to simplify permission management and enforce consistent access policies.
* **Established Role-Based Access Control (RBAC)**: Created IAM Roles with Trust Policies, enabling users and AWS services to securely assume temporary permissions (Assume Role).
* **Applied the Principle of Least Privilege**: Designed and assigned permissions that granted only the minimum access required for each user or role, reducing security risks.
