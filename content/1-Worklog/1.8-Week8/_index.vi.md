### Kết quả đạt được tuần 8:

* **Nắm vững kiến trúc Serverless trên AWS**: Hiểu rõ cách thức hoạt động của AWS Lambda, API Gateway, Amazon S3 và DynamoDB trong việc xây dựng hệ thống Backend không cần quản lý máy chủ.
* **Triển khai Backend Serverless hoàn chỉnh**: Xây dựng thành công hệ thống Backend sử dụng AWS Lambda để xử lý logic, API Gateway để tạo REST API endpoints và DynamoDB làm cơ sở dữ liệu.
* **Quản lý dữ liệu với Amazon S3**: Cấu hình S3 Bucket để lưu trữ tệp tĩnh, thiết lập Static Website Hosting và quản lý quyền truy cập cho các tài nguyên lưu trữ.
* **Thao tác cơ sở dữ liệu NoSQL với DynamoDB**: Tạo bảng DynamoDB, thực hiện các thao tác CRUD và tối ưu hóa truy vấn dữ liệu bằng Secondary Index.
* **Tích hợp Frontend với hệ thống Serverless**: Phát triển giao diện Frontend kết nối với API Gateway, gọi các Lambda Functions và hiển thị dữ liệu từ DynamoDB, hoàn thành luồng dữ liệu end-to-end.
* **Củng cố kỹ năng phát triển ứng dụng Cloud-Native**: Tích lũy kinh nghiệm thực tế trong việc xây dựng ứng dụng full-stack theo kiến trúc Serverless, sẵn sàng áp dụng vào dự án cuối khóa.
