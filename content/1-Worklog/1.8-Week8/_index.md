### Week 8 Objectives:

* Complete Lab exercises building Backend systems using Serverless architecture with **AWS Lambda**, **Amazon S3**, and **DynamoDB** database.
* Complete Lab exercises developing Frontend interfaces to connect and call APIs from the Serverless system.

### Week 8 Achievements:

* **Mastered Serverless Architecture on AWS**: Clearly understood how AWS Lambda, API Gateway, Amazon S3, and DynamoDB work together to build Backend systems without server management.
* **Deployed Complete Serverless Backend**: Successfully built a Backend system using AWS Lambda for logic processing, API Gateway for REST API endpoints, and DynamoDB as the database.
* **Managed Data with Amazon S3**: Configured S3 Buckets for static file storage, set up Static Website Hosting, and managed access permissions for storage resources.
* **Performed NoSQL Database Operations with DynamoDB**: Created DynamoDB tables, performed CRUD operations, and optimized data queries using Secondary Indexes.
* **Integrated Frontend with Serverless System**: Developed a Frontend interface connecting to API Gateway, calling Lambda Functions, and displaying data from DynamoDB, completing the end-to-end data flow.
* **Strengthened Cloud-Native Application Development Skills**: Accumulated practical experience in building full-stack applications using Serverless architecture, ready to apply to the final project.
