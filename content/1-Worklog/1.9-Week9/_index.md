### Week 9 Achievements:

* **Built Frontend Interface for the Project**: Designed and developed the main pages with reusable components, a complete routing system, and navigation.
* **Deployed Backend with Serverless Architecture**: Built Lambda Functions to handle business logic, designed REST API with API Gateway, and CRUD endpoints for each project feature.
* **Designed Optimized NoSQL Database**: Analyzed data requirements, created DynamoDB tables with appropriate key structures, and configured Global Secondary Index (GSI) to support complex queries.
* **Integrated Amazon Cognito Authentication System**: Set up User Pool for user management, configured secure Sign Up/Sign In workflows with MFA and strong password policies.
* **Completed End-to-End Authentication Flow**: Connected Frontend with Cognito, configured Identity Pool to grant AWS resource access, and successfully tested the entire registration, login, and authorization flow.
